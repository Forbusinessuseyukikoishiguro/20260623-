## 0. このツールは何か

査読デザイン論・写真構図・日本庭園の設計思想を「対話AI」で束ねて学び、さらに画像分析・キャラクター設計・レポート生成・AIパラメータ学習までを1ファイルで行う教材ツールです。CDN非依存・オフライン対応・APIキーはメモリ内のみ（保存しません）。鍵が無ければclaude.ai内のアーティファクト経路、鍵を入れればHTML経路で動きます。

## 1. タブ一覧

- 💬 **相談**：AIに質問。モード・ペルソナ・パラメータを設定。📎で画像添付。右に「🧪 くらべて学ぶ」学習デモ。
- 🌿 **デザイン論**：査読論文15本（実在DOI）。カードから相談を開始。

- 🎓 **学びデザイン**：問いのお手本＋ペルソナ色トーン（WCAG自動判定）。
- 🎬 **アニメ制作**：画像分析・キャラ辞書・数値表・100%再現プロンプト・利用者カスタムキャラ。

- ❓ **使い方**：丁寧なステップガイド。

## 3. プロンプト集の構成（100%再現）

各キャラに対し、以下のコピー可能プロンプトを収録します。色は実測HEX固定、造形は比率固定が再現の肝です。

- 基本プロンプト（日本語・推奨／English・汎用）

- ポーズ追加：上から（俯瞰）
- 表情編（12種・差し色つき：涙/汗 #99B7D2・怒り/舌 #D64E59・キラ #FAE078・頬 #FDCBCF）
- 背景：白 / ニューススタジオ
- ギャラリーシート（一覧）
- NG と対処
- 一貫性チェックリスト

## 6. 学習デモ「🧪 くらべて学ぶ」（エンジニアが学べる）

同じ質問を別設定で自動で投げ、出力の差を並べて見せます。1項目だけ変えて“効き”を体感する実験です。

- 🌡 **温度くらべ**：0.2（決定的）↔ 0.9（多様）
- 🪴 **専門度くらべ**：新人 ↔ エンジニア
- 🎭 **ペルソナくらべ**：査読者 ↔ 新人代弁者
- 🧭 **モードくらべ**：じっくり相談 ↔ 参照先
- 🎚 **3者くらべ**：庭師先生 / 査読者 / 新人代弁者
- 🧮 **2軸マトリクス（可変）**：温度・専門度・ペルソナ・モードから2軸を選び、各軸2段/3段で 2×2〜3×3 を一括比較（例：温度3×専門度2＝6通り）

**👍 これを保存**：結果カードから好みの設定を即反映し、**名前を付けて**⭐プリセット保存（温度・専門度・ペルソナ・モードを記憶）。
**⬇️ 書き出し**：くらべ結果を Markdown / HTML で出力（質問・**設定（温度・専門度・ペルソナ・モード）**・各ラベル・本文つき／自動でクリップボードにもコピー）。

## 7. AIパラメータ（変えるとどうなるか）

| パラメータ | 効果 |
|---|---|
| 温度 temperature | 低=決定的・再現性↑（出典重視・設計書レビュー）／高=多様・発想的 |
| 最大トークン max_tokens | 応答長の上限（HTML版 400–3000・アーティファクト版1000上限） |
| 回答モード mode | 参照先=即答＋DOI／考えさせる問／じっくり相談（四段構成）など |
| ペルソナ persona | 庭師先生／査読者／現場リーダー／新人代弁者で視点・口調が変化 |
| 詳しさ/専門度/たとえ/出典 | 自然言語の指示としてsystemに注入するプロンプト制御 |
| 話速 ttsRate | 読み上げ速度のみ（API非関与） |

設定を変えると「🔎 いまの設定だと→…」の効果プレビューがリアルタイムに変化します。良い設定はプリセットに💾保存、⭐で復帰、JSONで書き出し・読み込みできます。

## 8. 使い方（基本の流れ）

1. 💬相談タブで質問を送信。モード・ペルソナを選ぶと答え方が変わります。
2. 🔧パラメータを動かし、効果プレビューで挙動を確認（同じ質問で1項目だけ変えて投げ比べる）。
3. 「🧪 くらべて学ぶ」で温度/専門度/ペルソナ/モード/3者/2軸マトリクスを自動比較し、👍で好みを保存、⬇️で書き出し。
4. 📎で画像を添付し、🎬アニメ制作で造形・色彩・アニメの観点で精密分析→数値表＋100%再現プロンプト。
5. ➕で自作キャラを追加（背景・小道具つき）、エクスポートで保存。
6. 📄レポートで資料を15種のいずれかに整形。
7. 良い設定はプリセットに保存。困ったら↩︎で既定に戻す。
